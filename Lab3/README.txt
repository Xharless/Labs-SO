Detalles de uso del programa:
	- Para poder ejecutar el programa, debe tener en la misma carpeta archivo java, make y carpeta archivos_prueba.
	- Luego, para correr el código ejecutar el comando "make" en una terminal al mismo nivel de la carpeta con los archivos principales.
	

Consideraciones:
	- Nombre del archivo principal "Lab3.java". Nombre del archivo make "makefile". 
	- Para el correcto funcionamiento del programa, la carpeta "archivos_prueba" debe encontrarse al mismo nivel que el archivo Lab3.java
	- En caso de que se quiera seleccionar otra carpeta del tipo "Caso", este se debe cambiar manualmente en el codigo.
	- No es necesario que los archivos esten en un formato .txt, ya que en el formato que ya están funcionan correctamente
	- Se recomienda que si se quiere cambiar de carpeta para probar otro caso, se haga el comando "make clean"
	- Cuando se ejecuta el programa, en la salida por terminal se mostrara la ruta del archivo que se ejecetuto, tanto por Hebras como por Forks
	- Mostrara el tiempo que se demora tanto los Forks como las Hebras en encontar la letra (esta misma también se mostrara)
	- Una vez se ejecute todos los archivos, se entregara la clave completa y el tiempo total que se demoro en descifrarla por completo tanto para Forks y para Hebras
