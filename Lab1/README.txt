Detalles de uso del programa:
	- Debe tener en la misma carpeta los archivos "main.cpp", makefile y la carpeta "archivos_deportes", así funcionará correctamente el
	  código que está escrito en el archivo "main.cpp". La carpeta "archivos_deportes" debe contener los archivos .txt de prueba.
	- Para ejecutar el programa, abra una terminal en el directorio donde se encuentran los archivos descritos anteriormente, luego escriba:
	  "make run", el código se compilará y ejecutará. Al terminar verá que se creó una carpeta llamada "Carpeta Actual" que contendrá lo pedido en la tarea.

Consideraciones:
	- Se considera que está ejecutando el programa en un sistema Linux.
	- Los archivos de prueba están en la carpeta "archivos_deportes" y son .txt, además tienen el formato descrito en la tarea.

	- Sistema Operativo (SO): Ubuntu 22.04.4 LTS
